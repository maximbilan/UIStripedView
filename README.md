# UIStripedView

Simple class for creating striped view.

![alt tag](https://raw.github.com/maximbilan/UIStripedView/master/img/1.png)

## Installation
<b>CocoaPods</b>:
<pre>
pod 'UIStripedView'
</pre>
<b>Manual</b>:
<pre>
<i>Just copy the following files into your project:</i>

UIStripedView.h
UIStripedView.mm
</pre>
